# FitTrack 💚

Aplikasi tracker kesehatan harian — kalori, air, olahraga, dan berat badan.

## Cara Jalankan Lokal

```bash
npm install
npm run dev
```

Buka browser di `http://localhost:5173`

## Deploy ke Vercel

1. Push repo ini ke GitHub
2. Buka [vercel.com](https://vercel.com) → Import repo ini
3. Klik Deploy — selesai!
